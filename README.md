#Active Directory Sıkılaştırma Klavuzu
Bu dokümanda öncelikle active directory ile ilgili temel bilgiler verilmiş ve sonrasında sıkılaştırma için tavsiye edilen adımlar açıklanmıştır. Powershell Script'leri uygulanabilecek adımlar için yazılmıştır. Doküman hiç bir ticari amaçla yazılmamış olup bir çok kaynaktan yararlanılmıştır.
